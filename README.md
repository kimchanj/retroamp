## ✨ Features

- 🎵 MP3 playback (HTML5 Audio API)
- 🎚 Volume & Position control sliders
- ⏱ Real-time duration display
- 📂 File open dialog
- 📜 Playlist placeholder (PL)
- 🖥 Electron desktop build

## Run (dev)
```bash
npm install
npm run dev
